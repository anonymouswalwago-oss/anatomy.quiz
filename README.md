# Anatomy Quiz

A simple self-contained 100-question exam engine for mock anatomy practice.

Files of note
- `questions.html` — interactive 100-question exam UI (open in a browser).
- `index.html` — optional lightweight entry page that redirects/links to `questions.html`.

Usage
1. Open `questions.html` directly in your browser.
2. Or serve the repository locally (recommended) and open http://localhost:8000/questions.html:
   - Python 3: `python -m http.server 8000`
   - Node (http-server): `npx http-server -p 8000`
